# Open-Source PyMOL Wheels for Windows and Linux

This project builds Python wheels for release tags and arbitrary commits of the open-source version of PyMOL using GitHub Actions and makes them available on the [GitHub releases page](https://github.com/fhh2626/pymol-wheels-wl/releases).

- [PyMOL source code](https://github.com/schrodinger/pymol-open-source)
- [PyMOL open-source foundation](https://pymol.org/#opensource)
- [PyMOL license](https://github.com/schrodinger/pymol-open-source/blob/master/LICENSE)

Compared to the original [maabuu's PyMOL wheels](https://github.com/maabuu/pymol-wheels), I addressed a number of dependency issues, updated the dependencies to the latest packages, added support for Python 3.14, and fixed the Windows wheels.
